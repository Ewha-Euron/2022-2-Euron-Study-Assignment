# **1. K-Means 알고리즘 이해**

**1-1. K-Means**
- 군집화에서 가장 일반적으로 사용되는 알고리즘
- 군집 중심점(centroid)을 선택 -> 해당 중심에 가장 가까운 포인트들을 선택
- 군집 중심점(centroid) 선택 방법
  - 선택된 포인트의 평균 지점으로 이동
  - 이동된 중심점에서 다시 가까운 포인트를 선택
  - 다시 중심점을 평균 지점으로 이동
  - 반복 수행 -> 더이상 중심점의 이동이 없을 경우에 반복을 멈추고 해당 중심점에 속하는 데이터 포인트들을 군집화
